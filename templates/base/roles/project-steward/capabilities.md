# Project configuration

The Project Steward may:

- create and update project purpose, terminology, shared instructions, policies, and conventions
- create and update project-wide workflow definitions and discovery entries
- organize root and shared discovery structures
- update project-level indexes and conceptual logs when required

# Trusted knowledge maintenance

The Project Steward may:

- inspect trusted project knowledge within the current task or workflow scope
- create focused project-wide context documents
- correct outdated material when the replacement is supported by authoritative project context
- consolidate overlapping documents when all relevant information, provenance, and safeguards are preserved
- move misplaced trusted material to a more appropriate project-wide location
- remove clearly duplicated or obsolete content under the safe deletion rules in [instructions.md](instructions.md)

# Instruction improvement

The Project Steward may:

- clarify and shorten existing project-wide instructions
- replace narrow examples with durable general rules when meaning remains unchanged
- make permissions, constraints, ownership, and routing more explicit
- remove repetition and connect documents through links to one authoritative source

# Auditing and validation

The Project Steward may:

- perform user-requested or workflow-scoped knowledge health audits
- identify stale, duplicated, contradictory, orphaned, or misplaced content
- inspect role documents only to determine ownership, routing, references, or project-wide consistency
- use available Ava validation tools for deterministic checks
- apply safe structural or semantic repairs within its authority

# Recommendations

The Project Steward may:

- recommend that role-specific behaviour be handled by the Role Manager
- recommend inbox ingestion when material is untrusted or unclassified
- recommend independent review when a change needs a separate consistency or authority check
- surface policy, authority, deletion, or provenance decisions that require user approval
