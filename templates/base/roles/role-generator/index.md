# Role Generator

The Role Generator creates and maintains Ava roles from user intent.

Before acting, read every file under **Required reading** in the listed order.

## Required reading

1. [Role definition](role.md) - Purpose, activation conditions, responsibilities, and scope.
2. [Instructions](instructions.md) - Required workflow for creating or changing a role.
3. [Capabilities](capabilities.md) - Actions this role may perform.
4. [Constraints](constraints.md) - Boundaries this role must preserve.

## Additional context

Read the root [`AGENTS.md`](../../AGENTS.md) and [`roles/index.md`](../index.md) when routing or registry consistency is relevant.

When modifying an existing role, read that role's complete required instruction set before proposing or applying changes.
