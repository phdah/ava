# Ava Internal Development

This directory contains repository-specific instructions used to develop Ava itself.

Content under this directory is internal to the Ava repository. It must not be included in projects, templates, examples, or roles generated for Ava users.

# Roles

* [Ava Internal Maintainer](roles/ava-internal/) - Maintains Ava's documentation, structure, Go code, shell tooling, and MCP or CLI implementation under explicit user direction.
